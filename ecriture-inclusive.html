Guide de l'écriture inclusive

Comment écrire avec un clavier le point médian (·) utilisé dans l’écriture inclusive ?

La méthode dépend de votre système d’exploitation et du clavier que vous utilisez :

  • Si vous utilisez Windows, essayez la combinaison Alt + 250
  • Si vous utilisez Mac OS X, essayez une de ces combinaisons
    • ⌥ Alt + ⇧ Maj + f
    • ⌥ Alt + ⇧ Maj + h
    • ⌥ Alt + ⇧ Maj + 9
  • Si vous utilisez Linux avec un clavier Français, essayez une de ces combinaisons (ne pas utiliser le pavé numérique)
    • AltGr + :
    • AltGr + ⇧ Maj + ;
    • AltGr + ⇧ Maj + &
    • AltGr + .
    • AltGr + ⇧ Maj + .
    • AltGr + ⇧ Maj + h
  • Si vous utilisez Linux et qu’aucune de ces combinaisons ne marche, activez la touche "compose" et essayez la sequence suivante : Compose + . + -
